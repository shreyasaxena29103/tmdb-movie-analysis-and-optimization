# tmdb-movie-analysis-and-optimization
This project explores the TMDB movie dataset (900K+ records) using PySpark and Spark SQL in Databricks. It focuses on performance optimization and analysis

## Tools & Tech
- PySpark, Spark SQL
- Databricks (Community Edition)

##  Dataset
- Source: [TMDB Kaggle Dataset](https://www.kaggle.com/datasets/asaniczka/tmdb-movies-dataset-2023-930k-movies)
- Format: CSV (optimized to Parquet)
